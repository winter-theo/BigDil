# Mon Jeu Trop Bien

##Commande git utile :

```git clone``` :
pour copier un repo git en local

```git pull``` :
pour récupérer les derniers commits du repo distant

```git add *``` :
ajout de toutes les modifications de l'arborescence au futur commit

```git add nom_du_fichier```:
ajout du fichier en question au futur commit

```git status``` :
montre les différence avec le dernier commit

```git commit -m "message de commit" ```:
pour créer le commit

```git push``` :
pour pousser le commit sur le repo gitlab

```git revert``` :
pour revenir à un commit antérieur

```git branch``` :
pour connaitre la branche actuelle

```git checkout``` :
pour changer de branche

```git stash``` :
à utiliser quand il y a des conflits après un git pull https://git-scm.com/docs/git-stash/fr
